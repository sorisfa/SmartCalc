# 스마트 계산기 앱 개발 요청서 (Plan)

## 1. 목표
완전한 기능의 다기능 스마트 계산기 웹 애플리케이션 개발. 사용자가 일상생활에서 필요로 하는 다양한 계산 기능을 단일 인터페이스에서 직관적으로 제공합니다.

## 2. 핵심 기능 명세
1. **기본 계산기**: 사칙연산, 백분율(%), 괄호 연산 등 기본 수학 연산 및 초기화(C/AC) 기능
2. **날짜 계산기**: 특정 날짜 기준 D-day 계산, 기념일(100일, 1년 등) 계산, 두 날짜 사이의 일수 계산
3. **환율 계산기**: 한국 원(KRW), 미국 달러(USD), 일본 엔(JPY), 중국 위안(CNY) 등 주요 통화 간 변환 (실시간 환율 API 연동)
4. **부동산 평수 계산기**: 제곱미터(㎡) ↔ 평(坪) 양방향 자동 변환 기능
5. **이자 및 대출 상환 계산기**: 대출 금액, 금리, 기간을 입력받아 원리금균등, 원금균등, 만기일시상환 방식에 따른 월 상환액 및 총 이자 계산
6. **BMI 계산기**: 신장과 체중 입력 시 체질량지수(BMI) 계산 및 비만도 상태 시각적 표시
7. **연봉 및 실수령액 계산기**: 연봉 입력 시 4대 보험 및 소득세 등을 공제한 예상 월 실수령액 계산
8. **히스토리 기능**: 각 계산기별 최근 계산 기록(History)을 로컬 스토리지에 저장하여 재확인 가능
9. **반응형 디자인**: 모바일 스마트폰, 태블릿, 데스크탑 웹 브라우저 환경에 모두 최적화된 유동적 레이아웃 제공

## 3. 기술 스택
- **구조**: HTML5 (시맨틱 마크업)
- **스타일링**: CSS3 (Vanilla CSS, Flexbox/Grid, CSS Variables, Media Queries)
- **로직 및 동적 제어**: Vanilla JavaScript (ES6+, Fetch API, LocalStorage)

## 4. 개발 마일스톤
- **Phase 1**: 기획 및 디자인 명세(Plan.md, Design.md) 확정
- **Phase 2**: 공통 UI 프레임워크 및 반응형 레이아웃 퍼블리싱
- **Phase 3**: 핵심 계산기(기본, 부동산, BMI 등 로컬 연산) 로직 개발
- **Phase 4**: 심화 계산기(환율 API 연동, 대출, 연봉 등) 및 히스토리 기능 구현
- **Phase 5**: 크로스 브라우징 테스트, 디자인 폴리싱 및 최종 디버깅
